## 🌍 About Me

I'm a **.NET Engineer and AI/ML researcher** finishing my degree at **The Copperbelt University (CBU)**. My primary home is the **.NET ecosystem** — ASP.NET Core for backend APIs, Blazor for web UIs, and **.NET MAUI for cross-platform mobile and desktop apps**. I build end-to-end solutions entirely within the Microsoft stack, from database to UI.

My final year project is an **English ↔ Bemba Speech-to-Speech Translation System** — an end-to-end AI pipeline combining speech recognition, transformer-based NLP, and speech synthesis designed to improve accessibility for Zambian communities.

Beyond academia, I ship production software. I co-built two live fintech SaaS platforms — **FinHiveLoans** and **FinHivePOS** — currently serving real businesses in Zambia, built with Angular and ASP.NET Core.

I served as **President of the ICTAZ-CBU Chapter**, **Google Developer Student Clubs Lead**, and **Microsoft Learn Student Ambassador (Beta)** — roles that shaped my belief that technology only matters when it reaches people.

> *"Empowering people through technology, one line of code at a time."*
